# Campix Campus Management System

A C++ campus management system that allows users to view availability and reserve study halls, computer labs, and parking lots. The project also includes a companion encryption utility that secures user credentials using a Caesar cipher.

## Features

- Menu-driven campus reservation system
- View available study halls, computer labs, and parking lots
- Reserve campus spaces
- User credential handling through `.dat` files
- Caesar cipher password encryption utility
- Uses C++ inheritance, pointers, STL vectors, file parsing, and object-oriented design

## Technologies Used

- C++
- Visual Studio
- STL vectors
- File I/O
- Object-oriented programming

## Project Structure

- `campix-system/` - Main campus reservation system
- `encryption-utility/` - Password encryption utility

## How to Run

1. Open the `.sln` file in Visual Studio.
2. Build the project.
3. Make sure the `.dat` files are in the same directory as the executable.
4. Run the program from Visual Studio or the terminal.
